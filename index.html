Research Areas

Graph Neural Networks and Clustering

Developing self-supervised GNN architectures for unsupervised graph clustering and community detection in complex networks using deep learning and contrastive learning approaches.

Incremental Data Mining Algorithms

Designing adaptive incremental clustering algorithms for dynamic datasets including BISDBx, iMass, and BiMass using density-based approaches for real-time data stream analysis.

Adaptive Anomaly Detection Systems

Creating intelligent outlier detection frameworks using kernel density estimation and grid-based methods for identifying anomalies in evolving data streams and large-scale systems.

Research Work

G-DMoNLite: A Novel Modularity-Based Approach for Graph Clustering

Ongoing

Advancing unsupervised graph clustering through stable GNN architectures. Developed G-DMoNLite for efficient community detection with improved training stability and reproducibility.

Multi-Scale Diffusion Enhancement for Graph Clustering

Ongoing

Advancing graph clustering through multi-scale diffusion processes using heat and PPR kernels to capture hierarchical network structures effectively.

Attributed Modularity Loss for Improved Graph Clustering

Ongoing

Developing modularity-based loss functions that jointly optimize graph structure and node attributes. Created attributed modularity loss to improve clustering quality by integrating topological and feature information in neural graph clustering frameworks.
